# Countdown Slideshow

A simple countdown slideshow built using HTML, CSS, and JavaScript.

## Features

- Countdown timer from 10 to 0
- Automatic image slideshow
- Pure Vanilla JavaScript (no libraries)

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```
.
├── assets/
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.

## Preview

The slideshow displays a sequence of images while simultaneously running a countdown timer from 10 to 0.
